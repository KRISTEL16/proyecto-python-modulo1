# Proyecto Módulo 1 - Python for Analytics
## Sistema de Gestión Financiera Integral

**Descripción:**
Aplicación desarrollada en Streamlit que integra conceptos fundamentales de programación para la gestión de actividades financieras.

**Estructura del Proyecto:**
* **Ejercicio 1:** Verificador de presupuesto con condicionales.
* **Ejercicio 2:** Registro de actividades usando listas y diccionarios.
* **Ejercicio 3:** Cálculo de retornos con programación funcional (Map y Lambda).
* **Ejercicio 4:** Análisis de datos mediante Programación Orientada a Objetos (POO).

**Instrucciones de ejecución:**
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `streamlit run app.py`
